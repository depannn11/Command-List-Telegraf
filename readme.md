```

/chat        -> Chat AI
/gpt         -> GPT Response

/ailogo      -> Generate logo AI
```

**INSTALLATION**
```
npm install telegraf axios node-fetch form-data adm-zip js-confuser fs path os moment
```
==================================================

**CONFIGURATION**

Buat file config.js
```
BOT_TOKEN   = YOUR_BOT_TOKEN
ADMIN_ID    = 123456789
OPENAI_KEY  = sk-xxxx
API_KEY     = your_api_key
```

==================================================

**RUN BOT**

node index.js
npm start
